Example 133: EPU Flux - Multi-Particle¶

In this example we calculate the multi-particle flux near the peak on a rectangular surface 30 [m] downstram from the center of the EPU. This is meant for data that was ran on OSG or similar condor-type systems.

In [1]:
#Importing everything you will need
%matplotlib inline
import oscars.sr
from oscars.plots_mpl import *
import numpy as np
import glob

In [2]:
#Create a new OSCARS object
osr = oscars.sr.sr()
In [3]:
#After importing all of your files from OSG, this will create a list of all of the file names with the same name
file_list = glob.glob('osg_Example_133_*.dat')
In [4]:
#Sort the files so they are in order from 0 to 49
file_list.sort()
In [5]:
#Create a file list containing all but the first data file
a = file_list[1:]
In [6]:
#Average the files contained in list a
flux_a = osr.average_flux(ifiles=a)
In [7]:
#Create a file list containing just the first data file - this is the 'ideal' data
b = glob.glob('osg_Example_133_0.dat')
In [8]:
#Averages the file contained in list b
flux_b = osr.average_flux(ifiles=b)

Plot Flux¶

Plot the multi-particle flux. We'll pick a point around 154 [eV].

#Plot the ideal flux
plot_flux(flux_b)
In [10]:
#Plot the average flux
plot_flux(flux_a)
